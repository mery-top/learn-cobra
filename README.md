# learn-cobra

A learning repository for building CLI applications in Go using **Cobra** and related tools like **Viper**, **flag completion**, and **Stringer**.


### 🛠 mycli

A sample CLI application built with Cobra.

**Covers:**
- Commands and subcommands
- Flags
- CLI structure and organization
- Basic command execution flow

---

### ⚙️ Viper integration

Configuration management using Viper.

**Covers:**
- Reading config files (YAML / JSON / ENV)
- Binding flags to configuration
- Environment variable overrides
- Centralized config handling

---

### 🔎 Flag completion

Shell auto-completion for CLI flags.

**Covers:**
- Bash and Zsh completion setup
- Command and flag suggestions
- Improving developer experience (DX)

---

### 🧵 Stringer

Using Go’s `stringer` tool for enums.

**Covers:**
- Generating `String()` methods automatically
- Working with typed constants
- Cleaner logs and debugging
